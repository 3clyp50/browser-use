## Overview

OpenAI's **O3 model** provides the highest reasoning capability for browser automation tasks. This guide shows how to configure browser-use for maximum accuracy when precision is more important than speed.

<Note>
  O3 model is more expensive than other models, but provides superior accuracy for complex tasks requiring careful reasoning and precise execution.
</Note>

## Quick Setup

```python
from browser_use import Agent, BrowserProfile, ChatOpenAI

# O3 model with accuracy-optimized settings
llm = ChatOpenAI(
    model="o3",
    temperature=0.0,  # Minimize randomness
)

# Accuracy-focused browser profile
browser_profile = BrowserProfile(
    minimum_wait_page_load_time=1.0,
    wait_between_actions=1.5,
    slow_mo=500,  # Slow down for precision
    vision_detail_level='high',
)

# Create agent with accuracy settings
agent = Agent(
    task="your precise task here",
    llm=llm,
    browser_profile=browser_profile,
    max_actions_per_step=3,  # Lower for deliberate execution
    use_thinking=True,
    step_timeout=300,  # 5 minutes per step
    extend_system_message=ACCURACY_PROMPT,
)
```

### 1. Model Configuration

```python
llm = ChatOpenAI(
    model="o3",
    temperature=0.0,  # Deterministic responses
    # O3 specific settings for accuracy

```

**Key settings for O3:**
- `temperature=0.0`: Eliminates randomness for consistent results
- `model="o3"`: Uses the most capable reasoning model

### 2. Browser Profile for Accuracy

```python
browser_profile = BrowserProfile(
    # Extended wait times for complex pages
    minimum_wait_page_load_time=1.0,        # Wait longer before capturing
    wait_for_network_idle_page_load_time=2.0, # Ensure network completion
    maximum_wait_page_load_time=10.0,       # Allow time for heavy pages

    # Enhanced visual analysis

    # Browser stability settings

    slow_mo=500,                          # 500ms delay between actions
    timeout=60000,                        # 60 second operation timeout
    default_navigation_timeout=45000,     # 45 second navigation timeout
)
```

### 3. Agent Settings for Maximum Precision

```python
agent = Agent(
    task=task,
    llm=llm,
    browser_profile=browser_profile,
    
    # Deliberate execution settings
```

## Complete Example

```python
import asyncio
from browser_use import Agent, BrowserProfile, ChatOpenAI
from dotenv import load_dotenv

load_dotenv()

ACCURACY_PROMPT = """
Accuracy instructions:
- Analyze each page carefully before acting
- Verify element selections are correct
- Cross-reference extracted information
- Double-check all data for completeness
- Focus on precision over speed
"""

async def max_accuracy_task():
    # O3 model configuration
    llm = ChatOpenAI(
        model="o3",
        temperature=0.0,
    )
    
    # Accuracy-optimized browser settings
    browser_profile = BrowserProfile(
        minimum_wait_page_load_time=1.0,
        wait_for_network_idle_page_load_time=2.0,
        wait_between_actions=1.5,
        slow_mo=500,
        headless=False,
        vision_detail_level='high',
        highlight_elements=True,
        viewport_expansion=800,
    )
    
    # Create accuracy-focused agent
    agent = Agent(
        task="Extract company information from website with 100% accuracy",
        llm=llm,
        browser_profile=browser_profile,
        max_actions_per_step=3,
        use_thinking=True,
        flash_mode=False,
        max_failures=5,
        retry_delay=15,
        llm_timeout=120,
        step_timeout=300,
        use_vision=True,
        vision_detail_level='high',
        extend_system_message=ACCURACY_PROMPT,
        calculate_cost=True,
    )
    
    # Execute with conservative step limit
    history = await agent.run(max_steps=15)
    
    return history

# Run the accuracy-focused agent
asyncio.run(max_accuracy_task())
```

## Troubleshooting

### Common Issues

**Timeouts with O3:**
- Increase `llm_timeout` to 180+ seconds
- Increase `step_timeout` to 400+ seconds

**Element Detection Issues:**
- Set `vision_detail_level='high'`
- Increase `viewport_expansion` to 1000+
- Enable `highlight_elements=True`

**Inconsistent Results:**
- Set `temperature=0.0`
- Use specific, detailed prompting
- Increase `wait_between_actions`
